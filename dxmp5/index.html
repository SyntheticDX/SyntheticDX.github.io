DXMP Projects

These newer engines generally feature some bare-bones shooter template that with the latest UE5 is becoming increasingly fleshed out in form of the Lyra game.

Our dev content is modular as advised, nestled as a plugin inside the overall Lyra project that can be updated whenever.
